# Contributing to A'dan Z'ye AudioCodes

First of all, thank you for showing interest in this educational project!

This repository is designed as a structured educational book. To maintain the quality and consistency of the content, we follow a strict contribution process.

## How Can You Help?

### 1. Reporting Issues
If you find a typo, a broken technical example, or an outdated menu path for v7.20, please open an **Issue**. 
*   Specify the file name.
*   Describe the error clearly.
*   Suggest a fix if possible.

### 2. Suggesting New Topics
If there is a specific AudioCodes scenario you'd like to see added to the book, feel free to open an Issue with the tag `[New Topic Request]`.

### 3. Pull Requests
Direct Pull Requests are currently **restricted** to ensure the integrity of the educational flow and the digital watermarking system. If you wish to contribute a significant amount of content, please contact the owner first.

## Code of Conduct
*   Be respectful in discussions.
*   Focus on technical accuracy and educational value.
*   Respect the proprietary nature of this content.

Thank you for being part of the AudioCodes community!
